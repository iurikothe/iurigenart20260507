# Ideas

## Three rules (locked in)

Every piece in this project must satisfy all three:

1. **Responsive** — works at any screen size; layout uses relative units; no fixed pixel assumptions.
2. **Animated** — never a still image; the work moves on its own.
3. **Interactive** — at least a few keys (or pointer events) shape the live experience.

The thumbnail ArtBlocks captures is a single deterministic frame of the seeded
state, so interactivity never changes what gets minted — it only changes how
the piece feels when someone sits with it.
